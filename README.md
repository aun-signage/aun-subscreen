# aun: the kaigi signage system

## Configuration

You need to configure the following environment variables to start aun-subscreen:

## Required

* PORT
* DATABASE_URL
* TWITTER_AUTH
* TWITTER_QUERY
* IRC_SERVER
* IRC_PORT
* IRC_NICK
* IRC_CHANNELS

## Optional

* NODE_ENV
* TWITTER_EXCLUDE_REGEXP
* TWITTER_EXCLUDE_SCREEN_NAME
